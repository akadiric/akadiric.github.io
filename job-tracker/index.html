Job Tracker for Technical Graduates

Project Description

An application for technical graduates to track job applications. Users can create accounts to input their resume information, which is then used to generate tailored resumes and cover letters for each job they apply to. These AI-generated documents are available for download in .docx format, allowing users to edit them at a later time.

Technologies Used

The app is built with Next.js and styled with CSS, using OpenAI and Python to generate tailored resumes and cover letters.

Tools and Skills: NextJS, Javascript, CSS, OpenAI API, Python
