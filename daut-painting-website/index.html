Daut Painting Website

Project Description

The client wanted to refresh their website, DautPainting.com, design while adding new features and pages. To better showcase their work, I created a prominent "Recent Projects" section on the homepage, making it the first thing users see. I also added a masonry-style image gallery to showcase projects, allowing users to easily filter images by specific service categories. To highlight their transformation process, I integrated a before-and-after image slider, allowing visitors to compare projects pre- and post-painting. The client requested a minimalistic approach with a focus on simplicity for their website.

Technologies Used

The website is built with Wordpress and a custom PHP theme, styled with CSS to meet the client's design needs. Javascript is being utilized for image slider and lightbox functionality.

Tools and Skills: Wordpress, PHP, CSS, Javascript
