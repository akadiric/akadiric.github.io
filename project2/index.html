Justice Data and Design Lab Website and Reddit Bot

Project Description

The Justice Data and Design Lab wanted to move away from the typical, formal law firm website and create something that reflects their innovative lab. Our team redesigned the site with a fresh, modern look to make the legal world feel more approachable. It was important to them that Canadians are able to easily access available resources for legal guidance, so the website was designed with 'access to justice' in mind. Building on this idea of accessible knowledge, we integrated an AI chatbot into their website, JusticeDataAndDesign.com, that helps users find relevant resources for their legal inquiries (though not actual legal advice).

Technologies Used

The OpenAI API and Python were utilized to create the chatbot, trained on multiple PDFs containing legal information for BC residents. Combined with Javascript and CSS, we created a user interface for the bot, while the rest of the website was created using Wordpress and custom PHP themes.

Tools and Skills: Wordpress, OpenAI API, Python, Javascript, PHP, CSS/SCSS, HTML, Figma
